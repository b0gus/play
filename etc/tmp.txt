* Innovation sandbox : Explore new projects that showcase the potential for creativity, then bring new ideas and code to your own AI innovations.

- Microsoft Math(Cognitive)
: OCR(Optical Character Recognition) 기술로 사진 속의 수식 인식, 문제를 풀고 답과 해법을 도출

- Celebs Like Me(DNN)
: Facial recognition으로 사진 속 얼굴을 인식하고, 유사한 유명인들을 매칭함
: Bing Satori Knowledge Graph와 Bing Image Graph를 통해 학습

- Pix2Story(Conversational AI, DNN, NLP)
: 사진을 스캔하고 작문 스타일을 적용한 후 이야기를 생성 - 창의적 활동을 하는 AI에 대한 연구
: 이야기 생성에는 NLP(Natural Language Processing) 사용

- Snip Insights(Cognitive)
: 사진(스크린샷)에서 다양한 정보를 도출해내는 도구, 다운로드해 사용 가능
: 이미지에서 텍스트 추출 및 번역, 이미지 속 인물/장소/사물 자동 태그 등

- Machine Reading(ML, DNN)
: MRC(Machine Reading Comprehension)은 문서(텍스트)를 이해하고 이에 대한 질문을 답함

- Style Transfer(Cognitive)
: 사진을 통해 새로운 이미지들을 생성해내고, 새로운 스타일을 입힘(카메라 필터와 유사)

- Sketch2Code(Cognitive)
: 손으로 그린 그림을 HTML(Hyper Text Mark-up Language), 즉 웹 페이지로 변환

- Gen Studio(Cognitive, AI for Good)
: 저명한 미술작품들을 살펴보는 과정에서 AI가 작품들을 추천 및 변형해 새로운 경험을 할 수 있도록 함
: GAN(Generative Adversarial Networks) 활용

* Security and responsibility : Discover how we are applying our AI principles to create solutions that empower privacy, security, transparency, and inclusiveness.

- Differential privacy(Responsible AI, ML)
: 프라이버시를 위해 개인정보를 보호하기 위한 머신러닝 기술
: 머신러닝을 위해서는 데이터가 많을수록 좋지만, 데이터가 많을수록 프라이버시를 침해할 가능성도 높아짐. 이러한 문제 해결을 위한 연구.
: 통계적인 노이즈를 데이터에 추가해 개인정보를 덮으면서도 데이터의 질은 유지하는 컨셉트
: 실습(체험)은 불가능

- Homomorphic Encryption(HE)(Responsible AI)
: 데이터가 암호화 된 상태에서 계산을 수행할 수 있도록 하는 기술
: 컨텍스트를 그대로 유지하면서 암호화된 데이터셋으로 머신러닝을 수행할 수 있음
: Microsoft SEAL(Simple Encrypted Arithmetic Library) API 사용

- Responsible Conversational AI(Conversational AI, Cognitive)
: Conversational AI - 기업들이 고객들과 상호작용한느 새로운 방법으로 SNS, 챗봇 등 다양한 채널 포함
: 이러한 봇(bot)들이 사람들의 신뢰를 얻을 수 있는 방안을 모색하는 연구
